Изменение базы данных с Postgres на MySQL:

1. запрос создания таблицы изменить в соответствии с синтаксисом MySQL:
  - имя таблицы - public.funny_farm  ->  funny_farm
  - в FarmControl::$tableName - изменить public.funny_farm  ->  funny_farm
  - в FarmControl::__constructor() - при создании обьекта PDO изменить dsn ресурса 'pgsql' на 'mysql'

---------------------

Все запросы обрабатываются через index.php

По пунктам задания:

- размножение МРС происходит с периодичностью 10 секунд, если в загоне имеются минимум две овцы.

- AJAX - обновление страницы по нажатию кнопки 'Refresh'. Все остальные запросы - отправлются через POST.
- локализация: изменение языка по добавлению GET-параметра lang. На данный момент реализованы ru и en языки.
  Локализация описывается в farm/lang/<language>.php файлах соответственно.
- команды для перемещения и удаления овец. Принимаются на языке локализации или на английском. 
  Правила локализации команд описываются в farm/lang/<language>_commands.php файлах.
